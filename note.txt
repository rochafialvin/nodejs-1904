HTTP Status Code

2xx (Success)
200 : Ok. represent success
201 : Created. whenever new instance is created.
204 : No content. request is successfully processed with no returned data

4xx Error (Client)
400 : Bad Request. send the wrong data
401 : Unauthorized. indicate client is not allowed to access recources
403 : Forbidden. request is valid, client not allowed to access recources
404 : Not Found. recources it not available

5xx Error (Server)
500 : Internal server error. indicate request is valid. somethid bad happen in server
503 : Service Unavailable. server is down
